Registration Entry Requirements {#registration-entry-requirements} ================================================================== A registration entry, called metadata, has the following requirements: 1. Each metadata is defined as a separate {{VideoFrameMetadata}} dictionary member. 2. Each metadata MUST be serializable. 3. Each metadata should have clearly defined semantics. In particular, its interactions with the media processing pipeline (encoders, decoders, renderers...) should be well defined.
